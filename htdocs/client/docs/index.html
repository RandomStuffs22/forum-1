    8 декабря 2003 г.: вышла в свет версия 1.102

  • на панели инструментов добавилась кнопка "Start", по нажатии на которую
    выводится статистика по текущему форуму
  • появилась возможность задания интервала времени для отображения сообщений
    (заданные интервал не сохраняется между сеансами, при каждом новом сеансе
    интервал устанавливается равнм последним 30 дням)
  • реализована механизм открытия внутрфорумных ссылок без зачитывания данных
    из интернета
  • используются форумные иконки для маркировки сообщений

    25 ноября 2003 г.: вышла в свет версия 1.101

  • Исправлены некоторые проблемы, связанные с особенностями Invission Power Board - движка Исходников.RU

   28 сентября 2003 г.: вышла в свет версия 1.005
  • существенно ускорена загрузка программы
  • решена задача внутренних ссылок. Отныне не надо ходить в интернет, чтоб прочитать другое сообщение из форума. Проблема может быть только в случае отсутствия на локальном компьютере сообщений, на которые сделаны ссылки
  • сохраняется признак последнего зачитывания сообщений (маркируемых в дереве сообщений буквами N и U)
  • сделан инсталлятор
  • начата адаптация клиента под другие форумы (конкретно - GeekLog), так что вебмастера, желающие прикрутить клиента к своим форумам, милости прошу к сотрудничеству
  • исправлена ошибка: во избежание неудачного завершения программы теперь при добавлении в базу данных всегда заполняется поле иконки
  • исправлена ошибка: при задании цвета фона/шрифта вместо отображения сообщения отображалась информация, что запрашиваема страница не найдена
  • если главное окно приложения было минимизировано при закрытии, то при повторном запуске окно раскрывается на весь экран
  • после использование пункта меню "Show full branch" выводилась ошибка об отсутствии файла

    1 сентября 2003 г. : вышла в свет версия 1.004

  • исправлена ошибка: в меню использовалось "Readed", а не "Read" (стыдно мне, как автору программы изучения неправильных глаголов допускать такие ошибки, но исправлено)
  • в диалоге задания периода обновления добавлен флажок "Больше не показывать этот диалог". При следующем обновлении диалог будет показываться только если этот флажок снят или если в задается диапазон дат. Диалог всегда можно вызвать, использую пункт меню "View/Update period"
  • добавлен еще один акселератор "Ctrl-F" для поиска
  • ускорена загрузка программы (буду еще оптимизировать)
  • добавлена возможность отметить все сообщения в текущем форуме как прочтенные
  • при каждом запуске программы (а также при успешном ее завершении) удаляются временные файлы, которые могли оставаться от предыдущих сеансов
  • реализовано сохранение/восстановление местоположения главного окна приложения и ширины панели заголовков сообщений
  • исправлена ошибка: форум не отображался в списке, если из него не было зачитано ни одного сообщения
  • добавлен еще один акселератор "F5" для чтения сообщений из форума (работает только тогда, когда активна панель заголовков)
